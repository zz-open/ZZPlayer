# web-music-play
一款沉浸式网页音乐播放器

由 [HeoMusic](https://github.com/zhheo/HeoMusic) 修改而来，感谢原作者的辛勤付出！！！

原项目是html5+js实现，本人改进为vite+vue3+ts实现。

# 如何部署
本项目采用github pages部署(白嫖，嘻嘻！)，其他平台均可。

将构建后的dist文件内容上传到服务器即可。

自定义域名[可选]
